# Atividade HTML Semântico

## Descrição

Este projeto foi desenvolvido como parte de uma atividade prática com foco em **HTML5 semântico**, sem utilização de CSS. O objetivo é estruturar corretamente páginas web utilizando as principais tags semânticas.

O projeto contém dois layouts:

- **Wireframe de uma pizzaria**
- **Wireframe de uma loja de roupas**

## Como Executar

1. Baixe ou clone este repositório  
2. Abra os arquivos HTML em qualquer navegador:  
   - `pizzaria_wireframe.html`  
   - `loja_wireframe.html`  
3. Certifique-se de que as pastas **imagens/** e **wireframes/** estejam corretamente posicionadas

## Tecnologias Utilizadas

- HTML5 Semântico:
  - `<header>`
  - `<nav>`
  - `<main>`
  - `<section>`
  - `<article>`
  - `<footer>`
- Inserção de imagens com `<img>`
